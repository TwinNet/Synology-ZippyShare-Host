# Synology-ZippyShare-Host
Host Module for Synolgoy Download Station

##Installation
- Download the zippyshare.host
- Goto Download Station -> Preferences -> Filehosting
- Add the zippyshare.host

##Compile
The *.host is just a renamed zip File
